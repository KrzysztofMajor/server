Introduction

This weppage is constructed as part of the asignment A1, for the course Web Technology of the bachelor Computer Science at the VU in Amsterdam.

Everyone uses the Web, but how was it originally built, how has it evolved to what it is now, and how might it further develop in the future? The course will introduce the students to the key ideas, languages and protocols underlying the Web, including URI, HTTP, HTML, CSS and JavaScript. You will develop the skills you need to apply these techniques to create basic Web content and Web applications yourself, and to systematically validate and assess more advanced Web content made by others.

On this webpage, numerous famous authors will be showcased with their picture, appropriate tags, alts and a description
